# AI Resume Optimizer

A web application that uses AI to analyze and optimize 
resumes to help job seekers improve their chances.

🔗 Live Demo: https://ai-resume-optimizer-eight.vercel.app

## About
AI Resume Optimizer is a fullstack web application that takes 
your resume as input and provides AI-powered suggestions to 
improve it. Built with HTML, CSS, JavaScript on the frontend 
and Node.js on the backend, deployed on Vercel.

## Features
- Upload and analyze your resume
- AI-powered resume suggestions and improvements
- Clean and simple user interface
- Fast response powered by AI API
- Deployed and accessible online

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- AI Integration: AI API (Gemini / OpenAI)
- Fonts: Poppins
- Deployment: Vercel

## How to Run Locally

1. Clone the repository
   git clone https://github.com/ganesh18h/AI-Resume-optimizer.git

2. Go to project folder
   cd AI-Resume-optimizer

3. Install dependencies
   npm install

4. Create a .env file and add your API key
   API_KEY=your_api_key_here

5. Start the server
   node server.js

6. Open browser and go to
   http://localhost:3000

## Important
Never push your .env file to GitHub.
Your API keys should always remain private.
